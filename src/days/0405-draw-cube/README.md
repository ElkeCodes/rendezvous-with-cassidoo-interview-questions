# Draw cube

## Interview question

https://buttondown.com/cassidoo/archive/imagination-is-more-important-than-knowledge/

Write a function that draws an ASCII art cube of given height n.

Example:

> drawCube(2)
"
  +----+
 /    /|
+----+ |
|    | +
|    |/
+----+
"

> drawCube(4)
"
   +--------+
  /        /|
 /        / |
+--------+  |
|        |  |
|        |  +
|        | /
|        |/
+--------+
"
