# Max range

## Interview question

https://buttondown.com/cassidoo/archive/you-dont-have-to-be-great-to-start-but-you-have/

Given an array of integers, find the maximum range between two elements in the array.

Examples:
> maxRange([2,2,1,3])
> 2

> maxRange([0])
> 0

> maxRange([4,3,5,7,1,7,7,6])
> 6
